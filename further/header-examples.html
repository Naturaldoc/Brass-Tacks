Code

HTML

<header class="header-Base"> <div class="header-Content"> ... </div> </header> <header class="header-Hide"> <div class="header-Content"> ... </div> </header> <header class="header-Fixed"> <div class="header-Content"> ... </div> </header> <header class="header-Fixed-Shrink"> <div class="header-Content"> ... </div> </header>

CSS

.header-Base{ background:#446CB3; border-bottom:1px solid #333; box-shadow:inset 0 0 50px rgba(0,0,0,.2); color:#fff; margin-top:30px; position:relative; text-shadow:0 1px 1px rgba(0,0,0,.3); } .header-Fixed{
@extend .header-Base; // edit height for your own header size height:147px; left:0; margin:0; position:fixed; top:30px; width:100%; z-index:10; } .header-Fixed-Shrink{
@extend .header-Fixed;
&.shrinky{ font-size:70%!important; height:110px;   }
} .header-Hide{ @extend .header-Fixed; z-index:3; }

jQuery

$(window).bind('scroll', function(){ // checks to see if the window has scrolled more than 50px if($(this).scrollTop() > 50) { // if so, adds class 'shrinky' (feel free to use better class names) $(".header-Fixed-Shrink").addClass('shrinky'); } if($(this).scrollTop() < 50) { $(".header-Fixed-Shrink").removeClass('shrinky'); } });

This example is an extension off from the Brass Tacks framework.
